# **App Name**: Athkari

## Core Features:

- Display Athkar: Display daily Athkar (invocations) from a predefined list.
- Mark as Complete: Allow users to mark Athkar as completed for the day.
- Progress Indicator: Provide a progress indicator for daily Athkar completion.
- Daily Reminders: Set up daily reminders for Athkar times.
- Personalized Recommendations: A suggestion tool which helps user identify gaps in their Athkar.

## Style Guidelines:

- Primary color: Soft, muted green (#A7D1AB) to evoke a sense of peace and tranquility.
- Background color: Very light desaturated green (#F0F4EF).
- Accent color: Warm, desaturated yellow (#E4C493) for interactive elements and highlights, to give the feeling of a warm, welcoming sunrise.
- Clean and readable typography to ensure the text is easy on the eyes and aids in reflection.
- Simple and elegant icons representing different Athkar categories, using line art for a gentle and unobtrusive feel.
- Minimalist layout to reduce distractions, focusing on clarity and ease of use. Utilizing whitespace to allow the content to breathe.
- Subtle and calming animations for completing Athkar and receiving reminders.
